# Mobile Automation Tests

### Clone Project 

```
git clone https://github.com/Hrqlv/mobileTests.git
```
### Install dependencies

```
npm install / npm i
```

## How run the tests

### Running tests of SwagLabs
```
npm run wdio:swagLabs
```
### Running tests of ToDoList
```
npm run wdio:toDoList
```
